# Outputs Folder

This folder contains generated outputs from the R scripts.

## Folders

- `tables/` — CSV tables generated from the R scripts
- `plots/` — PNG plots generated from the sensitivity-analysis script

## CSV Outputs

The `tables/` folder contains:

### Sensitivity-analysis outputs

- `sensitivity_beta.csv`
- `sensitivity_eta.csv`
- `sensitivity_gamma.csv`
- `sensitivity_lambda.csv`
- `sensitivity_tm.csv`

### Crisp-model outputs

- `crisp_model_newton_iterations.csv`
- `crisp_model_computed_result.csv`
- `crisp_model_reported_vs_computed.csv`

### Fuzzy-model outputs

- `fuzzy_model_newton_iterations.csv`
- `fuzzy_model_computed_result.csv`
- `fuzzy_model_reported_vs_computed.csv`

## Plot Outputs

The `plots/` folder contains PNG plots showing the effect of parameter changes on:

- `t1` and `T`
- `GC(t1,T)` and `Q`

## Reproducibility

These outputs can be regenerated by running:

```r
source("code/sensitivity_analysis_data.R")
source("code/crisp_model_cost_function.R")
source("code/fuzzy_model_cost_function.R")
```
